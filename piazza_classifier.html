Piazza Machine-Learning Post Classifier

Description: A C++ text-classification tool that automatically labels Piazza discussion-board posts by topic.

Class: EECS 280 – Programming & Data Structures

Tools Utilized: C++, STL, Xcode / clang++

Project Duration: Nov – Dec 2023

My Role: Solo developer & algorithm designer (built on minimal starter code)

Project Requirements: View specifications


Problem Statement

Piazza threads for large courses quickly become cluttered. Students struggle to locate answers and instructors lose time responding to duplicate questions. An automatic classifier can streamline this process, eliminating extra time spent by both students and professors.


About the Course

EECS 280 is Michigan’s foundational C++ course. After CS introductory course, students tackle larger projects that combine data structures, algorithms, and clean software engineering in real-world scenarios.


Behind the Execution

I first re-implemented Binary Search Tree and Map classes from scratch to learn their internals, then swapped to the STL unordered_map for performance in the final classifier. The core class trains on a CSV using a bag-of-words model: word order is ignored, but token frequencies per label are tracked. It outputs log-prior and log-likelihood values in debug mode and predicts labels for unseen posts with a log-probability score.


Running from the Command Line

This program takes in several different arguments: a training file, a testing file and the option of having a "debug mode" activated. The training and test files contain a topic from a Piazza post (online question platform) and the words utilized in the post. If debug mode is enabled, it provides more information about the training data. How it was Executed: The project guidelines had us begin by creating our own Binary Search Tree and Map classes to gain a better understanding about how the two worked. However, for efficiency in the main classifier, we used the map library included with C++.


Code Distribution & Final Results

The classifier achieved ≈ 90 % accuracy on the blind dataset and scored 100 % on the course autograder. Course policy forbids publishing code derived from the starter kit, but I can demo the project on request.
