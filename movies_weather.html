Movie & Weather API Data Analysis

Description: A Python data analysis project exploring relationships between film release success and weather conditions in Ann Arbor using multiple APIs.

Class: SI 206: Data-Oriented Programming

Tools Utilized: Python, SQL, JSON, Matplotlib, Rotten Tomatoes API, OMDB API, IMDB API, OpenWeather API

Project Duration: November – December 2023

My Role: Team member in a group of 3 — independently handled API data cleaning, merging, and correlation analysis

View Code: GitHub Repository


Problem Statement

This project aimed to explore whether weather conditions could be a factor in movie popularity by comparing Ann Arbor’s weather the day of a movie release to box office success and movie ratings. We used API data to find patterns in how weather on release day might influence how well movies are received.


About the Course

SI 206 is the University of Michigan’s introduction to Python programming and data-driven development. Students learn how to work with APIs, databases, and data visualization libraries to solve real-world problems using clean, modular code.


Data Sources & Analysis

I accessed four APIs: one for current and historical weather data, and three for movie ratings and metadata to attempt to limit any potential bias. I wrote scripts to clean, merge, and store this data in a SQLite database, using SQL queries to join weather with movies released on that date. After processing, I used Python’s Matplotlib to visualize the data and analyze correlations.


Correlation Between Duration and Rating

Key Findings

  • Weather had minimal correlation with user rating or box office success — however, cold temperatures slightly aligned with higher rated movies.
  • Movie duration had a weak positive correlation with rating, though outliers were present.
  • Data analysis required significant preprocessing due to differences in formatting and API data structure.

Final Result

The project successfully demonstrated the ability to use multiple APIs, clean and store data efficiently, and perform basic statistical analysis with visualizations. You can read the full README here for more technical implementation details.
